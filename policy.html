Privacy Policy

Last updated: July 8, 2025
We collect no data whatsoever.

What this extension does

Adds a search option to your right-click menu. That's it.

What we don't collect

  • Browsing history
  • Personal information
  • Search queries
  • User activity
  • Analytics
  • Cookies

How it works

When you select text and use the context menu, it opens a new tab with a search URL. Everything happens locally in your browser.

Permissions

The extension needs minimal permissions to function:

  • Context menus — to add the menu option
  • Active tab — to read selected text

These permissions don't enable data collection.

Technical note

We have no servers, no databases, no analytics. There is nothing to collect data with.
